Environment
===========

Shared environment.

## `Environment([options])`

- `options`: initial options
- `extensions`: extensions
- `output`: output object
- `scripts`: scripts
- `services`: services
- `settings`: settings object
- `variables`: variables object
- `Logger`: passed logger initiator
